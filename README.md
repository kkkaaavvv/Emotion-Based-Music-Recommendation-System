# 🎧 Emotion-Based Music Recommendation System

A real-time web application that detects user emotions through facial expressions and recommends personalized music playlists.

---

## 🚀 Overview

This project uses computer vision and machine learning to analyze facial expressions captured through a webcam and classify user emotions. Based on the detected emotion, the system recommends a corresponding Spotify playlist tailored to that specific mood.

---

## 👩‍💻 My Work

This project was independently designed and developed end-to-end, including:

* Implemented real-time emotion detection using computer vision
* Integrated webcam input and facial expression analysis
* Built logic to classify multiple emotions using a pre-trained model
* Designed and developed the frontend interface for user interaction
* Created and mapped one Spotify playlist for each emotion
* Integrated playlist display into the UI for real-time recommendations

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** OpenCV
* **Model:** Pre-trained emotion classification model
* **Frontend:** Web-based interface

---

## ⚙️ How It Works

1. Capture user face via webcam
2. Detect facial features using OpenCV
3. Classify emotion using a trained model
4. Map detected emotion to a predefined playlist
5. Display the corresponding Spotify playlist

---

## 🎭 Supported Emotions

The system currently recognizes the following emotions:

* Happy
* Calm
* Sad
* Fear
* Anger
* Neutral
* Surprise

---

## 🎵 Playlist Recommendation Logic

Each detected emotion is mapped to a dedicated Spotify playlist created based on the mood associated with that emotion.

For example:

* Happy → Upbeat / energetic playlist
* Calm → Relaxing / chill playlist
* Sad → Slow / soothing playlist
* Anger → High-energy / intense playlist

This ensures a consistent and personalized listening experience.
